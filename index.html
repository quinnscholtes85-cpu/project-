Step-by-Step: How to Build a PC

1. Gather Your Components

CPU, GPU, RAM, Motherboard, Case, Power Supply, SSD/HDD, Fans.

2. Prepare the Case

Remove side panels, install standoffs, and prepare cables.

3. Install the CPU

Open the socket lever, align CPU correctly, and lock into place.

4. Install RAM

Align RAM sticks and firmly press into the slots until they click.

5. Mount the Motherboard

Carefully place it into the case and secure it with screws.

6. Install Storage

Attach SSD/HDD to the case trays or motherboard slots.

7. Install GPU

Insert into the PCIe slot and screw it into place.

8. Connect Power Cables

Supply power to the motherboard, GPU, storage, and fans.

9. Boot and Install OS

Start your PC, enter BIOS, and install Windows or Linux.
